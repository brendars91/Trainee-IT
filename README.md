# Trainee-IT

## Projektbeschreibung
Dies ist ein Übungsprojekt, um die grundlegenden Funktionen von Git und GitHub zu erlernen. Der Fokus liegt auf der Synchronisation zwischen lokalen und Remote-Repositories.

## Inhalt
- **README.md**: Projektdokumentation.
- **notizen.txt**: Einfache Textdatei für Übungsnotizen.
- **.gitignore**: Konfigurationsdatei zum Ausschließen von Dateien.

## Zweck
Das Ziel dieses Repositories ist es, den Workflow von Commits, Pushes, Pulls und das Lösen von Merge-Konflikten in einer realistischen Umgebung zu trainieren.
